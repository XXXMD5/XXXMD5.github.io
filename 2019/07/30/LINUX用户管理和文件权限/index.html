LINUX用户管理和文件权限

2019-07-30

用户管理

/etc/passwd 用户信息
/etc/group 组信息
/etc/shadow 密码相关

root:x:0:0:root:/root:/bin/bash
用户名 x uid gid 名字 家目录 登录后启动的脚本

​ root:x:0:
​ 组名字 x gid 组员

​ root:$6$Xf7Mz580:17750:0:99999:7:::

​ 用户名 密码 。。。

useradd 添加用户 useradd zs

userdel 删除用户
userdel -r zs 删除用户所有信息

usermod 修改用户信息
usermod -s /sbin/nologin 锁定登录脚本
usermod -d /home/123 修改家目录
usermod -G -a root zs 把zs用户加入到root组

groupadd 添加一个组

groupdel 删除一个组

groupmod 修改一个组
groupmod -g 123 zs 修改zs组的gid为123
groupmod -n ls zs 修改zs组的名字为ls

passwd 修改密码
passwd root 修改root的密码

文件权限

-rw-r–r–

文件类型

  • 普通文件
    d 文件夹/目录
    b/c 驱动等硬件设备
    l 链接文件

文件权限

​ r 针对文件 读/可以查看文件内容 针对文件夹 读/可以列出问价夹里面的内容
​ w 针对文件 写/可以修改文件内容 针对文件夹 读/可以创建文件
​ x 针对文件 读/可以执行 针对文件夹 读/可以切换到目录

​ rwx => 421

​ chmod 777 a.txt
​ chmod a+x a.txt

a/u/g/o  +/-/=  r/w/x
655
-R      //修改文件夹下所有的文件

​ chown apache:apache a.txt
-R 修改文件夹下所有的文件

三部分 : - rw-r–r–

​ 第一部分 前三个 针对于所属用户
​ 第二部分 中间三个针对于所属组
​ 第三部分 最后三个针对于其他用户
