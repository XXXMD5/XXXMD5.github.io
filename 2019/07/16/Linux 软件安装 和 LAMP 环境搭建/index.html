Linux 软件安装 和 LAMP 环境搭建

2019-07-16

软件安装

三种安装方式

LAMP环境搭建

搭建环境
apache
yum install httpd httpd-devel httpd-menaul mod_ssl mod_perl mod_auth_mysql -y
service httpd start
vi /etc/httpd/conf/httpd.conf

service httpd restart
chkconfig httpd on
netstat -antlp 查看80端口是否打开，用curl访问一下，curl 127.0.0.1

mysql
yum install mysql-server mysql -y
chkconfig mysqld on
service mysqld start
netstat -antlp //查看3306端口是否打开
mysql_secure_installation

php
yum install php php-mysql php-gd libjpeg* php-imap php-ldap php-odbc php-pear php-xml php-xmlrpc php-mbstring php-mcrypt php-bcmath php-mhash libmcrypt -y

service httpd restart
